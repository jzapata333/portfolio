Synamatic | Reseña de Películas

Este proyecto es inspirado por Netflix, en el que puede ver las clasificaciones de películas recién estrenadas y próximas. Incluye enlaces para ver los tráilers de las películas que te interesan. En este proyecto encontrará un directorio de categorías, funciones de búsqueda y la capacidad de guardar sus favoritos también como películas que todavía no ha visto.

Este proyecto utiliza React-Redux y Django API.

Dream House

Dream House es un sitio web de anuncios de bienes raíces donde puede buscar casas para la venta y alquiler, así como enviar su solicitud para vender su casa con Dream House. Puede guardar propiedades en su favoritas una vez que se haya registrado para una cuenta. Tambien puedes reiniciar tu sesión para ver tus propiedades guardadas.

Este proyecto utiliza React-Redux y Django API.
